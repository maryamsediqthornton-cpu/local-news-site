Important:
- This ZIP deliberately excludes .git, so it cannot change your GitHub branch.
- A ZIP/package cannot switch branches by itself.
- Branch changes/conflicts happen when GitHub Desktop is already on a branch or mid-merge.

Safe deployment:
1. In GitHub Desktop, make sure there is NO merge conflict open.
2. If there is, click Abort merge.
3. Stay on your intended branch: V005_FinalBranding.
4. Unzip this package.
5. Open the local-news-site folder inside it.
6. Select everything inside that folder.
7. Drag into Documents/GitHub/local-news-site and replace files.
8. Commit once and push.
